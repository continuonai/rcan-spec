# RCAN v1.6 Implementation Tracking
**Version:** 1.6.0  
**Status:** IN PROGRESS  
**Date:** 2026-03-16  
**Source gaps:** GAP-14, GAP-16, GAP-17, GAP-18 (deferred from v1.5)

---

## Scope

v1.6 addresses the four medium-complexity gaps deferred from v1.5 due to spec size constraints. All four gaps are now required for multi-org deployments, remote/IoT robot deployments, and richer data collection workflows.

## New MessageType Integers (v1.6)

No new message type integers are added in v1.6. The existing types are extended:

| Type | Name | v1.6 Change |
|------|------|-------------|
| 9 | DISCOVER | Response payload gains `supported_transports` field |
| 10 | TRAINING_DATA | JSON-only payload deprecated; `media_chunks[]` required for image/video/audio |
| 12 | FEDERATION_SYNC | Full payload spec defined (was placeholder in v1.5) |
